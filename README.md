# Login System Testing (Manual QA Project)

## 📌 Project Overview

This project demonstrates manual testing of a Login functionality, focusing on validating user authentication and input handling.

---

## 🧪 Test Coverage

* Valid login scenarios
* Invalid username and password combinations
* Empty input fields
* Input format validation (email/username rules)
* Error message verification
* Boundary and edge cases

---

## 📝 Test Case Details

* Total Test Cases: 15+
* Includes positive, negative, and edge scenarios
* Designed using structured test case format

## 📂 Files
